### Week 4 Objectives:

* Understand the architecture and core concepts of Amazon Simple Storage Service (Amazon S3).
* Learn how to create and manage S3 buckets and objects.
* Practice configuring bucket permissions, versioning, and lifecycle management.
* Understand best practices for storing and securing data on Amazon S3.

### Week 4 Achievements:

* Developed a comprehensive understanding of Amazon Simple Storage Service (Amazon S3).

* Learned the core concepts of Amazon S3, including:

  * Buckets
  * Objects
  * Object Keys
  * Prefixes
  * Storage Classes
  * Metadata

* Successfully created and managed Amazon S3 buckets.

* Practiced common object management operations, including:

  * Uploading objects
  * Downloading objects
  * Copying objects
  * Moving objects
  * Deleting objects

* Understood different Amazon S3 Storage Classes, including:

  * S3 Standard
  * S3 Intelligent-Tiering
  * S3 Standard-IA
  * S3 One Zone-IA
  * S3 Glacier Instant Retrieval
  * S3 Glacier Flexible Retrieval
  * S3 Glacier Deep Archive

* Successfully configured Bucket Policies and IAM Policies to secure S3 resources.

* Understood the differences between:

  * Bucket Policy
  * IAM Policy
  * Access Control List (ACL)

* Enabled Bucket Versioning and practiced restoring previous object versions.

* Configured Lifecycle Rules to automatically transition and expire objects, helping optimize storage costs.

* Learned AWS best practices for securing Amazon S3, including:

  * Blocking Public Access
  * Applying the Principle of Least Privilege
  * Encrypting data at rest
  * Using Versioning for data protection

* Gained practical experience in managing cloud storage for web applications and enterprise systems.

* Improved troubleshooting skills by resolving common bucket permission and object access issues.

* Documented all laboratory exercises and deployment procedures for future reference.

* Established a solid foundation for integrating Amazon S3 with other AWS services such as EC2, RDS, Lambda, and CloudFront in future projects.
